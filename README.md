# 什么是Lunar CN

关于Lunar CN

Lunar CN全名Lunar Client for China

Lunar CN 基于Lunar Client修改,增加了汉化和饰品破解等,其本质是一个Minecraft启动器

Lunar Client官网:<mark style="color:yellow;">https://lunarclient.com</mark>
